# RENACE HATS – Catálogo Web Corporativo

## Descripción
Sitio web corporativo tipo catálogo para una marca de sombreros artesanales colombianos.
Enfocado en la presentación visual de productos, experiencia de usuario y optimización SEO para mejorar la visibilidad en buscadores. Publicado con dominio propio y alojado en Netlify.

## Tecnologías
- Frontend: HTML, CSS, JavaScript
- SEO: Estructura semántica, meta etiquetas y optimización para motores de búsqueda
- Hosting: Netlify

## Enlace
🔗 https://renacehats.com/

## Funcionalidades
- Catálogo de productos con visualización clara y organizada
- Diseño responsive adaptado a dispositivos móviles y escritorio
- Optimización SEO básica para indexación en Google
- Navegación intuitiva orientada a la presentación de la marca
- Diseño enfocado en la identidad visual artesanal

## Aprendizajes
- Implementación de estructuras semánticas para mejorar SEO
- Optimización de rendimiento en sitios estáticos
- Diseño centrado en experiencia de usuario (UX)
- Despliegue de proyectos con Netlify
- Desarrollo de sitios web corporativos para clientes reales

## Nota
Este proyecto fue desarrollado durante una etapa temprana de mi aprendizaje en desarrollo web utilizando HTML, CSS y JavaScript puro. Aunque actualmente aplico tecnologías modernas como React, Next.js y TypeScript, he decidido mantener este repositorio porque representa mi evolución como desarrollador y continúa funcionando correctamente en producción.
