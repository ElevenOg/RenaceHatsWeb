# RENACE HATS – Catálogo Web Corporativo

## Descripción
Sitio web corporativo tipo catálogo para una marca de sombreros artesanales colombianos.  
Enfocado en la presentación visual de productos, experiencia de usuario y optimización SEO para mejorar la visibilidad en buscadores. Publicado con dominio propio y alojado en Netlify.

## Tecnologías
- Frontend: HTML, CSS, JavaScript
- SEO: Estructura semántica, meta etiquetas, optimización para motores de búsqueda
- Hosting: Netlify

## Demo
🔗 https://renacehats.com/

## Funcionalidades
- Catálogo de productos con visualización clara y organizada
- Diseño responsive adaptado a dispositivos móviles y escritorio
- Optimización SEO básica para indexación en Google
- Navegación intuitiva orientada a conversión y presentación de marca
- Sección de productos con enfoque en identidad visual artesanal

## Aprendizajes
- Implementación de estructuras semánticas para mejorar SEO
- Optimización de rendimiento en sitios estáticos
- Diseño centrado en experiencia de usuario (UX) para catálogos de productos
- Despliegue de proyectos con hosting en Netlify
- Mejora en la presentación visual de marcas reales con enfoque comercial
